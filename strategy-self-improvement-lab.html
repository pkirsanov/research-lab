Strategy Self-Improvement & Walk-Forward Lab

The viral “self-improving trading agent” loop — done honestly. Define an explicit numeric goal, run a transparent rule strategy, then let a scientific-method search change one variable at a time and keep only the changes that survive out-of-sample walk-forward scoring. The point of the tool is the guardrail the hype videos skip: an improvement only counts if it holds up on data the search never optimised on. 100% in-browser, deterministic synthetic paths — no live prices, no network, fully reproducible from a seed.

Educational research tool — NOT investment advice. Every price path here is synthetic: generated in-browser from a seeded random process with configurable regimes, so it is fully reproducible but has no connection to any real asset. The strategy is a deliberately simple, transparent rule chosen to make the method legible, not to be traded. Out-of-sample walk-forward scoring and the multiple-testing discount reduce — but never eliminate — overfitting; on real markets you also face costs, slippage, capacity, regime change and non-stationarity that this toy omits. The lesson to carry away is the process (explicit goal → change one variable → confirm out-of-sample → discount for search breadth), not any specific lever value. Do your own due diligence and size your own positions.
